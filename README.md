# 🏹 President Cupid's

A specialized matchmaking and social networking platform designed specifically for the **President University** community. This project aims to foster meaningful connections within the campus through a secure and verified environment.

## ✨ Key Features
- **Student-Only Verification**: Uses university-specific credentials to ensure a trusted user base.
- **Interest Matching**: Algorithm-based connections focusing on hobbies, majors, and campus activities.
- **Secure Messaging**: Private and safe communication channels for verified students.
- **Profile Customization**: Options to showcase personality, academic interests, and organization involvement.

## 🛠️ Technical Stack
- **Frontend**: HTML5, CSS3, JavaScript (Responsive UI)
- **Backend**: PHP
- **Database**: MySQL (Managing user profiles and connection logs)

## 🚀 Installation
1. Clone the repository to your local machine.
2. Import the SQL database file via phpMyAdmin.
3. Configure the database connection in the config file.
4. Run the project using a local server like XAMPP.
